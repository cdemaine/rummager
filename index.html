Drag and drop this to your bookmarks bar

Installing On Mobile

Use This Code snippet

Chrome Instructions
  1. Copy the bookmarklet code above.
  2. Bookmark this site.
  3. Click Bookmarks in the menu
  4. Long click the bookmark you just created.
  5. Paste copied code snippet into the URL field.
  6. Press Done
  7. You can now run this bookmarklet on the bevager site!
Safari Instructions
  1. Copy the bookmarklet code above.
  2. Bookmark this site.
  3. Open the bookmarks menu
  4. Tap Edit and select the bookmark you just created.
  5. Paste the bookmarklet code in place of the Address/URL field.
  6. You can now run this bookmarklet on the bevager site!

Usage

Chrome on Android
  1. Go to Bevager
  2. Click on the address bar
  3. Type in the name of your bookmark ("Rummager" if you drag and droppped it)
  4. Select the the bookmark from the search results. It should begin with "javascript:"
  5. Enjoy all the new search items
Everything Else
  1. Go to Bevager
  2. Open the new Rummager bookmark
  3. Enjoy all the new search items
